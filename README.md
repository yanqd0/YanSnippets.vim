# YanSnippets.vim

## Introduction

This is a collection of my personal general snippets in Vim.

## Dependency

Make sure the Vim plug-in [UltiSnips][1] is installed and accessable in Vim editor. If you use [Vundle.vim][2], it like this:

```VimL
Plugin 'SirVer/ultisnips'
```
